# 🌿 Plant Disease Prediction

![Dataset Samples](images/sample-leaves.png)

This project is an AI-powered plant disease prediction system developed using CNN and deep learning techniques. It detects plant leaf diseases from images with high accuracy using TensorFlow and image processing. This solution helps in fast and reliable disease identification for smart agriculture.

---

## 🚀 Features

- Detects plant diseases from leaf images  
- High accuracy CNN model  
- Supports 39 plant disease categories  
- Image preprocessing and classification  
- Fast and reliable predictions  
- Smart agriculture based solution  

---

## 🛠️ Technologies Used

- Python  
- TensorFlow / Keras  
- CNN (Convolutional Neural Network)  
- NumPy  
- Pandas  
- Matplotlib  
- OpenCV  
- Google Colab  

---

## 📂 Dataset

This project uses the **Plant Leaf Diseases Dataset with Augmentation** containing healthy and diseased leaf images.

- Total Classes: **39**
- Images: **61,486+**
- Training Split: **80%**
- Validation Split: **10%**
- Test Split: **10%**

### Dataset Source:
Mendeley Data

---

## ⚙️ Data Preprocessing

- Dataset extracted from ZIP file  
- Images resized to **160 x 160**
- Batch size: **32**
- Data split into train / validation / test sets
- Shuffling enabled for better training
